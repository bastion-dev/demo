# Bastion Demo API

This project consists of an API that can be used to write Bastion tests for during a demonstration. The API consists of a single `Sushi` resource which we can perform `POST` and `GET` requests on.

##Sushi

Sample `Sushi` entity.
```json
{
    "id" : 123,
    "name" : "Salmon Nigiri",
    "price" : 1.50
}
```
 
 ###POST `/sushi`
 
 ###GET `/sushi`
 
 ###GET `/sushi/{id}`
